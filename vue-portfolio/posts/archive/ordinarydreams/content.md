### Project Description

**Ordinary Dreams** is a public-space project in Robertsfors, a small city outside Umeå, designed to create an adaptable marketplace for its inhabitants. The project takes inspiration from electrical circuits, where the structure acts as a resistor guiding social interactions. Influenced by architects like Cedric Price and Fazlur Khan, the design prioritizes flexibility and responsiveness to user needs.

### Key Features

- Modular and reconfigurable structure
- Horizontally suspended frame supported by three vertical pillars
- Mobile crane system for structural transformation
- 2640 individual cubes for dynamic space creation
- Counterweighted suspension for structural stability

### Process

The project explored an adaptable marketplace concept where social interactions shape the built environment. The structure is controlled by users who reposition cubes to form temporary constructions, accommodating fluctuating community demands. Inspired by tension distribution in boats, a ring-based mediator system was designed to evenly distribute structural loads.

### Implementation

- **Structural System**: A suspended horizontal frame supported by three pillars with counterweights
- **Reconfigurable Modules**: 2640 movable cubes manipulated via an overhead crane
- **User-Controlled Adaptability**: Community-driven transformations for dynamic space usage

### Design Principles

The project emphasizes:
- **Adaptability**: Structure evolves with user needs
- **Participatory Design**: Users actively shape the space
- **Engineering Efficiency**: Counterweighted suspension minimizes structural constraints
