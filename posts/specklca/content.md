### Project Description

A opensource web-based LCA tool for architects and designers to make LCA calculations for buildings throughout the whole project life span. Done as a opensource project part through LINK Arkitektur and on off time. Built with direct connection to Speckle, a opensource datahub to be able to do quanity takeoffs from any software or workflow. 

The tool is built with a modular design principle, so that each module has clear roles in what data they modify and manipulate so each part could be substituted by internal or existing workflows. Allowing users to either do the whole process or just slot in at the very last steps and only do result viewing using Speckle.

Link to Github repository: https://github.com/linkarkitektur/SpeckleLCA

### Key Features

- Web-based interface
- Early phase LCA calculations
- Integration with architectural workflows
- Open source architecture
- Real-time updates on results

### Process

Development and design process includes:

1. Development of app framework and buisness logic
2. Design frameworks on UX and user workflow
3. Implementation of web interface and integration with Speckle
4. Open source release
5. Testing inhouse at LINK Arkitektur and other community members
